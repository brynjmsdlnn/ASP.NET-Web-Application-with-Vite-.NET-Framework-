# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this template, please report it privately and directly by email:

- **Email:** [bryanjamesdalanon@gmail.com](mailto:bryanjamesdalanon@gmail.com)

Include the following details so we can reproduce and assess the issue quickly:

- Affected component or file
- Steps to reproduce
- Impact and severity
- Any suggested mitigation or fix

## Scope

This repository is a starter template for:

- ASP.NET MVC/Web API 2
- ASP.NET Web Forms / Razor views
- Vite 8 frontend integration
- Tailwind CSS 4.2

We monitor and address reports as soon as possible.

## Acknowledgements

Contributors who responsibly disclose vulnerabilities may be credited in release notes when applicable.
